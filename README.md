## 安装

在本機安裝相關依賴

### Unix

```bash
# 主程式需要的相關套件
pip install SpeechRecognition pyaudio pygame xpinyin
# 安裝 FLAC，FLAC（Free Lossless Audio Codec）是一種音頻編碼格式
# Linux
sudo apt-get install flac
# macOS
brew install flac
# 確認是否已經安裝成功
flac
```
